Making a new initiatory

  1. Open up terminal
  2. Type git init myInit

You should get a message like:

Initialized empty Git repository in /home/cj/myInit/.git/

The name of your repo is important to remember. You can now go into your files and see that myInit is in your system.

Basic Navigation

(This is going to continue assuming that you have the terminal open and you're continuing from the last steps.)

  1. Type cd myInit

Your terminal should be something like cj@cj:~/myInit$

Congratulations! You are now in your directory for myInit.

If you want to make new files, continue to the next step. I am going to list how to change files if you have more than one folder in the directory.

  1. Type cd FILE

For example, we are now in cj@cj:~/myInit/FILE$. If we had an app.js file in the FILE folder, for instance, we could move from: cj@cj:~/myInit$, type cd FILE/app.js, and be in cj@cj:~/myInit/FILE/app.js$ without having to manually input moving forward one file at a time.

Now, if you want to go back ONE file, you would type cd ... That would land you in cj@cj:~/myInit/FILE$. To go back to the main file from ANYWHERE YOU ARE, type cd.
